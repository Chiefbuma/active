# **App Name**: Taria Health Compass

## Core Features:

- User Authentication: Secure authentication system for all users with role-based access control.
- Patient Self-Onboarding: Registration process for patients to create accounts and provide necessary health information.
- Personalized Marker Assignment: Assign relevant health markers (e.g., blood pressure, glucose levels) to patients based on their chronic disease and current week in the program.
- Progress Monitoring: Track patient's progress on assigned markers over time and generate reports.
- Personalized health advice generation: Using AI to provide recommendations about users health and their metric.
- Progress Summary Dashboard: Visualize user progress and display to user in form of dashboards

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust, stability, and health.
- Background color: Very light blue (#E8EAF6), near-white, to create a clean and calming atmosphere.
- Accent color: Soft teal (#4CAF50), a brighter color with less saturation, to highlight key metrics and actions with a sense of growth and progress.
- Body and headline font: 'PT Sans', a humanist sans-serif suitable for both headlines and body text, providing a modern yet warm feel for readability.
- Use health-related icons with a modern, minimalist style to represent markers, progress, and actions.
- Use a clean, card-based layout to organize information on the dashboard for easy navigation and understanding.
- Subtle transitions and animations to provide feedback on user interactions and guide the user through the app.
